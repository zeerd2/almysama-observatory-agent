# Almysama Observatory Agent

Lightweight Linux agent for Almysama Observatory.

Install from a token generated by the observatory:

```bash
bash -c 'set -o pipefail; wget -qO- https://raw.githubusercontent.com/zeerd2/almysama-observatory-agent/main/install.sh | sudo bash -s -- --server https://agent.almysama.tech --token <install-token>'
```

The installer detects Linux architecture, downloads the matching GitHub release asset, writes `/etc/almysama-agent/config.json`, registers a systemd or OpenRC service, and starts the agent.

Collected data:

- system uptime, load, memory, and root disk usage
- running systemd services
- Docker containers
- listening ports from `ss -tulpn`
- Caddy and Nginx version signals when available

The first request uses the short-lived install token. After enrollment, the server returns a per-agent secret and all heartbeat/report requests are signed with HMAC-SHA256.

Uninstall:

```bash
wget -qO- https://raw.githubusercontent.com/zeerd2/almysama-observatory-agent/main/uninstall.sh | sudo sh
```
